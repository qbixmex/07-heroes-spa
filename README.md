# Heroes App SPA

This is a Heroes Application from Marvel and DC Studios

## Install Dependencies

### YARN

```
yarn
```

### NPM

```
npm insall
```

## Development Mode

### YARN

```
yarn dev
```

### NPM

```
npm run dev
```

## Make a build

### YARN

```
yarn build
```

### NPM

```
npm run build
```

## Testing

### YARN

```
yarn test --watch
```

### NPM

```
npm run test --watch
```
